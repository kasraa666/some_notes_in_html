📖 导论：贯穿中华人民共和国对外关系的三条主线
🔴 主线一：中国革命运动
革命运动塑造政党身份——"独立自主"原则源自革命教训（"真正懂得独立自主是从遵义会议开始的"——毛泽东，1963）核心概念
革命经历形成基本世界观——"国际体系的本质就是压迫和不平等"核心认知
中共1949年前对外关系三阶段：1937年前共产国际支部 → 抗日民族统一战线时期 → 全面内战爆发后站在苏联阵营时间线
🏛️ 主线二：中国内政
"对外关系是内政的延续"——国家统一/领土完整、经济发展、社会核心价值构建、执政合法性核心命题
前三十年高度关注国家安全；改革开放以来关注和平发展阶段对比
国家发展战略对外交政策的影响具有滞后性——文革结束后出现矛盾现象（需要和平环境 vs 判断"苏修亡我之心不死"）高频考点
🌍 主线三：国际环境
冷战两极格局（1949-1991）→ 后冷战多极化；国际环境的客观变化 vs 领导人的主观认知认知框架
📎 核心概念：对外关系 vs 外交
对外关系涵盖范围更广：外交部（政治）+ 商务部（经济）+ 国防部（军事）+ 中联部（党际）制度认知
中联部是党中央机构（非国务院下属），负责中共对外关系——"党政双轨"对外关系体系易错点
📗 第一单元：建国初期的对外关系（1949-1955）
1.1 建国前的中共对外关系（1945-1949）
"蒋之下的和平"——美苏两大阵营 × 国共两大阵营的四方关系核心概念
美苏都支持"蒋之下的和平"——美国出于对美苏关系和抗日需要，苏联出于避免与美冲突和维护东北利益（1945年中苏友好同盟条约由国民党签订）1945
中共策略（1945）："在全国范围内大体要走法国的路"、"我们还是钻进去给蒋介石洗脸，而不要砍头"——毛泽东重要语录
斯大林初期对中共持怀疑态度——称中共为"人造奶油"共产党；转折点：1948年中共军事胜利后斯大林承认判断错误关键细节
1.2 中苏同盟的形成
米高扬西柏坡谈话——1949年1月1949.1
刘少奇秘密访问苏联——1949年6月1949.6
毛泽东访问苏联，签订《中苏友好同盟互助条约》——1949年12月至1950年2月，确立同盟关系，明确加入苏联阵营1949.12-1950.2
两大转折点：① 1945年底东北协调（开启合作第一步）② 1948年起中共军事胜利（地区性合作→全面战略合作）转折点
1.3 三大外交原则
"一边倒"——和苏联结盟 + 对帝国主义国家不承认。定位：总揽全局的方针核心原则
"另起炉灶"——不承认旧的外交关系，重新建立新的外交体系核心原则
"打扫干净屋子再请客"——清除帝国主义在华势力后再与西方建交核心原则
三大原则的启示：冷战中美苏两大集团关系的根本性质是冲突；中国领导人对"革命中建立的国家的认识"决定了对外政策走向论述必备
1.4 中美关系与抗美援朝
中美交恶原因：冷战背景 + 美国对中共的打压；并非历史的必然，但可改变历史的机会不多历史判断
军事目标的动态调整：10.2"歼灭和驱逐美国及其他国家的侵略军" → 10.3"恢复战争爆发前状况" → 10月中旬"构筑防御线"关键细节
"草鞋没样，边打边像"——毛泽东重要语录
意义：国家安全保障 + 民族自信提升 + 国际地位提高 + 现代化战争经验 + 台湾问题延宕五点意义
1.5 日内瓦会议（1954）
1953年苏联提出"五国外长会"，解决朝鲜问题 + 印度支那问题背景
奠边府战役大捷（1954.5.7）——印支讨论的关键转折点关键节点
两大矛盾点：停战问题（周恩来斡旋越共）+ 军事分界线（18度线还是13度线，周恩来再次斡旋）斡旋角色
意义：调整外交政策的开端 + 周边外交和邻国关系良好开端 + 开始与西方国家展开接触 + "五大国"身份初步确立四大意义
1.6 和平共处五项原则与万隆会议
和平共处五项原则：互相尊重主权和领土完整、互不侵犯、互不干涉内政、平等互利、和平共处必须背诵
1954.4 中印关于西藏的协定——五项原则的最初萌芽，首次写入"互相尊重领土主权、互不侵犯、互不干涉内政、平等互惠、和平共处"1954.4
周恩来"休会期之旅"（1954.6.25-6.29）：访问印度和缅甸，签署联合声明1954.6
1954.7.7中央政治局扩大会议：决定大幅调整对外政策——结束"打扫干净屋子再请客"1954.7.7
"门要关死已经不可能了，而又有一种很有利的局势，需要我们走出去。"——毛泽东，1954重要语录
毛泽东再提"中间地带"（1954，时隔八年1946→1954），意义：推动从革命外交向国家外交转变理论基础
万隆会议（1955）：第一次由亚非独立国家参加且没有西方殖民国家参加的大型国际会议；方针——"求同存异"核心考点
五项原则的三大意义：① 国家身份构建（社会主义+亚洲新兴国家）② 革命外交→国家外交转变的标志 ③ 破解美国对华孤立三大意义
印度的"中间人"角色：1950年代印度在中美之间扮演信息传递角色，尼赫鲁试图在中美之间发挥调解作用周边外交
1955年第一次台海危机：1955年1月解放军攻占一江山岛→美国国会通过《福尔摩沙决议案》授权总统使用武力→推动中美大使级会谈开启台海危机
📘 第二单元：发展和波动的十年（1955-1965）
2.1 中共八大与八项外交方针（1956.9）
重要判断：革命运动时代已经结束；社会矛盾发生变化；国际局势趋向缓和，世界持久和平开始有了实现的可能核心判断
八项外交方针：巩固与苏联关系 + 和平共处五项原则基础上同亚非国家建交 + 同愿意建交的国家发展正常关系 + 反对使用武力 + 支持和平运动 + 反对殖民主义 + 支持社会主义运动 + 反对大国主义八项内容
后来外交"纠左"时被重新回归——邓小平时代关于"和平与发展"的论断实质是重回八大判断前后呼应
2.2 苏共二十大及其影响（1956.2）
"三和路线"：和平共处、和平过渡、和平竞赛核心概念
赫鲁晓夫秘密报告——批判斯大林个人崇拜，引发社会主义阵营混乱（波匈事件）关键事件
中共不同看法：对和平过渡保留意见；对秘密报告不完全认同（应全面评价斯大林）；留下隐患分歧萌芽
苏共二十大召开后国际反应：东欧动荡、资本主义国家共产党内部混乱；社会主义阵营在"非斯大林化"问题上陷入分裂国际影响
2.3 波匈事件（1956）与中共介入
处理原则：对波兰——强调独立平等原则；对匈牙利——强调社会主义阵营稳定，排斥背离社会主义的倾向双重原则
影响：中共威望上升 + 斯大林评价分歧加深 + 中共认为应旗帜鲜明反对修正主义双重影响
赫鲁晓夫在波匈事件危机中"首先想到的就是'同兄弟的中国共产党协商'"——中共实际扮演了稳定社会主义阵营的角色关键细节
2.4 莫斯科会议（1957）
国际共运史上规模最大、人数最多的盛会；发表《莫斯科宣言》；中国在会议中具有特殊地位历史地位
既维护了团结，仍暴露了分歧——宣言是"表面团结、深层分歧"的结构辩证评价
2.5 1958-1959：不断扩大的矛盾
四大矛盾：① 长波电台事件（主权和使用权）② 联合舰队事件（军事控制问题）③ 炮击金门（对外政策分歧）④ 大跃进与苏联态度（发展道路分歧）四大矛盾
1959年赫鲁晓夫访华受到冷遇——标志着中苏同盟走向衰亡的转折点关键转折
中苏在国际问题上的分歧开始被归结为赫鲁晓夫背离马克思主义——"修正主义"标签出现定性转变
中苏在核问题上的矛盾：苏联推动禁止核试验谈判、推迟履行向中国提供原子弹模型的协定 → 1959年6月苏联单方面废止协议；中方认为苏联与美国妥协损害中国利益核分歧
"赫鲁晓夫莫名其妙！毛泽东是怎么啦？"——赫鲁晓夫就联合舰队事件反应（1958.7），反映了双方互信的严重断裂重要细节
庐山会议后毛泽东"对赫鲁晓夫的宣战"：彭德怀事件后，毛泽东将苏联对公社的批评与党内斗争联系起来——"内外呼应"；中苏矛盾与国内政治交织内外交织
2.6 1960-1965：中苏关系的破裂
阶段一（1960春-1961夏）：布加勒斯特会议→短暂缓和→第二次莫斯科会议时间线
阶段二（1961.10起）：苏共二十二大争执——"三和路线"升级版 + "全民国家""全民的党" + 再次批判斯大林1961.10
阶段三（1962起持续恶化）：伊塔事件 → 核不扩散条约 → 中苏论战 → "九评"苏共中央公开信关键节点
对破裂的反思：不同历史文化国情 + 对外关系不同看法（缓和 or 斗争）+ 国际形势判断不同 + 无产阶级国际主义 vs 国家间关系原则的冲突深层原因
苏共二十二大（1961.10）关键内容："全民国家""全民的党"理论；将斯大林遗体移出列宁墓——从根本上触犯了中共对社会主义本质的理解关键事件
十月革命47周年庆祝活动（1964）——中方确认苏联内外政策不因赫鲁晓夫下台而根本改变；中苏分裂已成定局最终确认
2.7 中印边界冲突详情（L7）
中印关系蜜月期：1954.4中印关于西藏的协定 → 1954.6周恩来访印 → 和平共处五项原则1954
"朗久"事件（1959）：中印边界首次武装冲突；苏联公开表态持中立态度令中国不满首次冲突
周恩来与尼赫鲁通信（1959年9-11月）：中方主张"应根据历史背景和当前实际情况，通过友好协商全面解决边界问题"；尼赫鲁坚称"麦克马洪线无可争议"外交交锋
1959-1962年：双方试图缓和的努力失败 → 1962年自卫反击战 → 中方主动后撤1959-1962
古巴导弹危机与中印战争同期（1962.10-11）：美苏聚焦古巴，为中印冲突提供了特殊的国际窗口国际背景
苏联因素：苏联在1959年中印冲突后保持中立→不满中国→1962年冲突前夕苏联加强向印度提供军事援助三角关系
2.8 中美大使级会谈详情（L7）
1954年日内瓦会议期间"领事级接触"：5月中旬美国通过英国渠道要求中国释放被押美国侨民；周恩来认为应"快开快谈"但未达成任何成果会谈前史
1955.8.1 中美大使级会谈正式开始——直接起因是1955年第一次台海危机，美国急于管控台海局势1955.8.1
钱学森回国（1955年）：1955.6钱学森通过亲属转信给陈叔通，中方以此要求美方放人；侨民回国协议是15年会谈中达成的唯一协议标志性成果
1957.12美方单方面降低会谈级别——中方视为破坏谈判→会谈中断；反映出美国内部在对华政策上的矛盾中断原因
2.9 其他重要内容
中印边界冲突（1962）：传统"习惯线" vs "麦克马洪线"（1914年西姆拉会议——中方从未承认）；古巴导弹危机同期发生事件
"两个中间地带"：① 亚非拉经济落后国家 ② 欧洲、北美加拿大、大洋洲等；美苏为"第一世界"；意义——为对外关系提供新落脚点，促成了第二次建交高峰理论演进
中美大使级会谈（1955-1970）：世界外交史上独特的"马拉松"会谈，共136次，唯一协议——侨民回国（钱学森回国）独特案例
中美大使级会谈三个阶段：① 1955.8-9（侨民协议）② 1955.9-1957.12（解除禁运+外长会谈，因美降级而结束）③ 1958金门炮战后（危机管理+沟通渠道）三阶段
📙 第三单元：文革期间的中国外交（1966-1976）
3.1 对外政策"左转"的前夜
"左"的含义：追求超越时代或高于现实可能性的目标；运用与实际能力不符的方式去追求目标概念界定
王稼祥与"三和一少"：主张和缓方针、争取和平国际环境（八大方针延续）→ 被批判为"对帝国主义、修正主义和各国反动派要和，对民族解放运动支持要少"关键人物
批判"三和一少"的后果：错过了一次调整对外关系和对外政策的重要机会，为文革极左外交铺路深远影响
"左转"前夜的复杂背景：1954-八大期间平稳发展势头破灭——对外（中苏同盟逐步破裂）+ 对内（大跃进和人民公社失败）→ 交织复杂的关系为"左转"奠定了基础背景分析
极左外交对外交系统的冲击时间线：1967.1上海造反派夺权→ 1967年5月外交部门开始受到冲击 → 1967.7-8外交部全面瘫痪时间线
3.2 极左外交思潮
八大十一中全会公报（1966.8）：世界革命新时代 → 帝国主义走向全面崩溃、社会主义走向全世界胜利 → 无产阶级国际主义是最高指导原则核心文件
极左外交四大特征：① 时代判断失误 ② 高估中国地位（"中国革命中心论"）③ 无产阶级国际主义至上 ④ "两个拳头打人"（反帝必反修）必须背诵
3.3 极左外交的实践
外交部夺权事件：王力"八七讲话"→"红卫兵就不能干外交？"→批斗外交部党委→外交部运作瘫痪关键事件
"三砸一烧"：冲击印度使馆 + 袭击缅甸使馆 + 冲击印尼使馆 + 火烧英国代办处（1967.8.22，最严重）。共同特点：均发生在外交部被冲击最严重的1967年7、8月极左顶峰
火烧英国代办处背景链：1967年5月香港劳资纠纷（左派工人示威）→ 香港当局镇压（逮捕新华社记者）→ 北京群众抗议→英国代办处被冲击纵火事件链条
极左外交其他表现：输出世界革命（在驻在国支持反政府运动）+ 宣传毛泽东思想（红宝书全球发行，突破"内外有别"原则）其他表现
3.4 中国安全战略重塑：苏联成为主要威胁
1960年起：零星边界冲突开始出现（伊塔事件等）；苏联开始向中苏边境增兵1960
中苏边界"三笔领土账"：① 沙俄时期通过不平等条约割占（《瑷珲条约》《北京条约》等）② 苏联继承并巩固的边界 ③ 外蒙古独立问题历史经纬
"北顶南放"战略：1950年代形成的国防方针——北方坚决顶住苏联方向压力，南方沿海灵活防御；前提是中苏同盟正常运作战略概念
1964年：毛泽东放弃"北顶南放"战略（6月）；首次公开提出中苏边界问题（7月接见日本社会党人士）；苏联以"核战争"回应；10月赫鲁晓夫下台后毛泽东开始担忧苏联军事入侵1964
1966年：苏蒙条约 → 苏联军事力量向中蒙边境推进1966
1968.8 苏联入侵捷克斯洛伐克：勃列日涅夫主义——"有限主权论""国际专政论""社会主义大家庭论"等 → 苏联从"安全威胁"进化为"新的战争策源地"关键转折
1969.3 珍宝岛事件：毛泽东明确提出"要准备打仗"；中国安全主要威胁由美国变为苏联核心转折
"四人小组"战略研判：苏联威胁比美国大；"在可预见的时期内，美帝、苏修单独或联合发动大规模侵华战争的可能性都还不大"；叶剑英："东联孙吴，北拒曹魏"战略建议
毛泽东1966年3月讲话："我们要准备修正主义来打，打进满洲，东三省，打进新疆，中央突破，从外蒙古打进北京……我们还要准备美苏合作，瓜分中国。"——体现了极端的安全焦虑重要语录
中苏边界全长约7300公里；50年代蜜月期"万里北疆有边无防"——边界问题被搁置；1960年代开始成为冲突焦点地理背景
3.5 中美和解
中国释放信号：转载尼克松就职演说（1969.1《人民日报》全文刊载）+ 提议恢复中美大使级会谈（1969.2）+ 两次试探性接触（1969夏、1969秋）1969
周恩来通过巴基斯坦传递关键信号（1969.12.2）：告诉巴基斯坦驻华大使——美国可以通过正式渠道与中国接触；标志着中国决策层已准备采取行动突破信号
尼克松方面的条件积累：美国内部出现调整对华政策声音 + 美国社会对华舆论扭转 + 尼克松作为"最让人放心的大鹰派"具有转向对华接触的独特政治信誉美方条件
乒乓外交（1971.4）：邀请美国乒乓球队访华——"小球转动大球"标志事件
基辛格秘密访华（1971.7 & 10）：讨论印支问题、中美关系正常化时间表、台湾问题（核心中的核心）关键步骤
尼克松访华（1972.2）：毛泽东当日会见；周恩来会谈全球战略、台湾、地区问题；发表《上海公报》历史时刻
中美和解意义：实现历史性和解 + 中国摆脱外交孤立 + 改变"战略三角"力量平衡，推动苏联削弱和解体及冷战结束三大意义
3.6 中美苏三角关系的复杂性
三方博弈：美国（联华遏苏但摆缓和姿态制约中国）+ 苏联（对华打和谈旗，对美国打中国牌）+ 中国（响应苏联边界会谈缓和局势，对美国打苏联牌加速中美正常化）三角博弈
毛泽东担心美国"踏着中国的肩膀跑莫斯科"——"被牵连"（Entrapment）的恐惧关键概念
⚠️ 中国调整政策的主要原因主要是外部安全环境，内生动力不强——这就是中美关系在尼克松访华后长期未突破的原因核心判断
三角关系的结构性演变：1950s 严格两极（无三角空间）→ 1960s 中苏分裂打破两极（有雏形但中国"两个拳头打人"未利用）→ 1970s 真正三角博弈 → 1980s 超越三角，中国成为独立一极结构演变
尼克松访华后中美关系低潮原因：尼克松后期（水门事件后外交停滞）+ 福特时期对苏缓和（台湾问题倒退）+ 卡特初期企图继续"缓和"政策→直到70年代后期缓和政策破产才迈开正常化步伐低潮分析
3.7 "一条线、一大片"与"三个世界"
"一条线"（1973.2，基辛格第五次访华）：从东到西建立反苏霸统一战线——美国、日本、中国、巴基斯坦、伊朗、土耳其、欧洲战略方针
"一大片"：联合"一条线"周边的国家——"一大片的第三世界要团结"战略延伸
"三个世界"理论：第一世界（美苏）+ 第二世界（欧洲、日本、澳、加等）+ 第三世界（广大亚非拉）；邓小平1974年联大讲话正式提出核心理论
"三个世界"意义：成为1970年代中国外交指导思想 + 奠定对第三世界关系的立足点 + "第二世界"概念扩大外交空间理论意义
3.8 重返联合国
董必武参加1945年旧金山制宪会议——中共是联合国创始参与者1945
1971年三个提案：① 美日"重要问题"提案（需2/3多数）② "两阿提案"（恢复席位+驱逐蒋代表）③ 美国"双重代表权"提案关键节点
2758号决议通过；中国起初曾准备推迟进入（凌青回忆录），但最终决定参加重要细节
意义：中国国际地位加强 + "一个中国"原则成为当代国际体系组成部分历史意义
📕 第四单元：改革开放以来的中国外交（1978至今）
4.1 文革结束与国家战略转移
1977.8 十一大宣布文革结束；1978.12 十一届三中全会——以经济建设为中心，实行改革开放1977-1978
国家战略和对外政策的根本关系：对外政策发生根本性转变的动力来自于国家战略的转变——以经济建设为中心 → 为经济发展创造良好外部环境核心逻辑
华国锋1977年政治报告："苏美两家是新世界大战的策源地，特别是苏联社会帝国主义具有更大的危险性"——体现滞后性滞后性例证
邓小平1981年关键讲话："世界大战在几年内打不起来…我们不要自己吓唬自己，造成人为的紧张。如果当前老是强调战争马上打起来，使我们的精力都集中到打仗上面，就会影响四个现代化建设。"重要语录
4.2 外交"纠左"
纠左三大核心内容：① 摆脱"时代"问题束缚——提出"和平与发展"是当今世界的主题（认识时代问题上的巨大飞跃；重回中共八大判断）；百万大裁军（1985）第一维度
② 对中国国际地位作出客观评价——"中国是个大国，又是个小国"（邓小平）；"千万不要当头，这是一个国策"第二维度
③ 国家利益回归对外政策的出发点——外交为经济建设服务、为现代化服务；从"国家性质"→"安全威胁"→"经济建设"第三维度
邓小平论和平时间："我们希望至少有50年到70年的和平时间。……我们面临发展和摆脱落后的任务。"发展逻辑
4.3 毛→邓外交理念转变
三维度对比：关于时代——毛"革命和斗争" vs 邓"和平和发展"；关于中国地位——毛"中国革命中心论" vs 邓"贫穷落后，需努力追赶"；关于决定因素——毛从"国家性质"到"国家安全" vs 邓"经济建设"论述题核心
4.4 中共十二大与"独立自主"（1982）
80年代三大任务：加紧现代化建设（核心）+ 实现国家统一 + 反对霸权主义和维护世界和平战略框架
外交方针大调整：不同任何大国结盟；在独立自主、和平共处五项原则基础上同一切国家发展关系；核心——独立自主核心内容
演变脉络："一边倒" → "一条线，一大片" → "独立自主"；里程碑意义的重大调整脉络对比
"中国不会打'美国牌'去对付苏联，不会打'苏联牌'去对付美国，也决不允许任何人玩'中国牌'"——黄华，1982年经典表述
4.5 中美建交
1978年布热津斯基访华 → 中美建交谈判 → 1978.12《建交公报》：承认PRC是唯一合法政府；只有一个中国，台湾是中国一部分；1979.1.1与台湾断交、结束共同防御条约1978-1979
中方动力升级：从单纯对抗苏联扩张 → 配合国家发展战略需要——"我们相信中美关系正常化能为美国用先进的东西帮助我们实现四个现代化创造更有利的条件"（邓小平，1979）动力转变
《建交公报》vs《上海公报》：《建交公报》前进一大步——美方承认PRC是"唯一合法政府"（而非仅"acknowledge"）；"承认只有一个中国，台湾是中国的一部分"；1979.1.1美台断交、终止《共同防御条约》公报对比
美国国会随即通过《与台湾关系法》（1979.4）——承诺继续向台湾提供防御性武器，保留对台湾问题的干预杠杆遗留问题
4.6 中苏关系正常化
1979年不再续约《中苏友好同盟互助条约》——同盟关系正式结束，之后进入30年条约有效期终止程序1979
1982年勃列日涅夫"塔什干讲话"释放缓和信号——表示愿意改善中苏关系、承认中国是社会主义国家；中方"投桃报李"但明确提出三大诉求1982
中方三大诉求：① 中苏边界和蒙古驻扎重兵 ② 支持越南入侵柬埔寨 ③ 武装占领阿富汗三大障碍
第一轮尝试（1982-1986）：成效有限（苏联领导人频繁更替，未在核心关切上让步）；第二轮（1987-1989）：恢复边界谈判 → 苏方撤军 → 1989年戈尔巴乔夫访华——30年后第一位苏联最高领导人访华1982-1989
"结束过去，开辟未来。……只是了解，不予纠缠，重点在于开辟未来。"——邓小平，1989重要语录
意义：中国走上独立自主的和平外交道路的一个重大标志历史意义
4.7 中国与联合国关系的变迁
三阶段：① 体制内的反对者（1971-1978）——反霸讲坛，保持超脱，"不参加"投票 ② 全面认识和务实对待（1978-1990s中）——服务于经济建设，全面深化参与 ③ 新时代——"参与者、建设者、贡献者、受益者"阶段框架
标志性事件：1975年UNDP在华设立代表处——中国开始将联合国视作获得发展资源的平台，而不仅是政治斗争讲坛转变标志
国际秩序话语演变：文革"帝国主义走向全面崩溃" → 1970s"三个世界反霸" → 1990s"建立国际政治经济新秩序" → 近年"维护以联合国为核心的国际体系"话语演变
